National Park Icons

Icon Collection

The Brief: The aim of this self-initiated project was to create a set of 80 generic icons to be used for signage and map-icons for American national parks. This particular brief was used as a exercise for myself to better my skills as a designer.

The Outcome: The result is a vast set of icons that are both playful and informative in appearance. The design and layout of each one carefully follows a strict grid to maintain consistency between icons. The set has been organised into the following catagories: Tourist and Recreation, Points of Interest, Services, Transportation, Prohibition, Saftey and Information.
